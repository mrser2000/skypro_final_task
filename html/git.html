Git

  • git add - добавляет файл или файлы в индекс, готовит их для коммита.
  • git commit - формирует новый коммит с комментарием, описывающим суть внесённых правок.
  • git log - отображает список коммитов в порядке от самых новых к более старым, позволяя проследить историю изменений проекта.
  • git branch. Управляет ветками в репозитории: создаёт новые ветки, а также просматривает, переименовывает и удаляет существующие.
  • git checkout. Используется для переключения между ветками или восстановления файлов в рабочем каталоге до состояния из определённого коммита или индекса.
  • git fetch. Загружает актуальные данные из удалённого репозитория в локальное хранилище без изменения файлов в рабочей директории.
